# Plantixia - Page de Présentation

Ce répertoire contient la page de présentation web de Plantixia.

## 🎨 Design & Technologies
- **HTML5 / CSS3** : Structure et styling (Vanilla).
- **Lucide Icons** : Pack d'icônes pour une interface moderne.
- **GSAP & ScrollTrigger** : Animations de reveal fluides lors du défilement.
- **Responsive Design** : Compatible avec Desktop, Tablette et Mobile.

## 🚀 Contenu
- **Hero Section** : Présentation du concept et vision.
- **Stats** : Indicateurs de performance de l'IA.
- **Features** : Détails sur le scanner, le support et la communauté.
- **Team** : Présentation de l'équipe fondatrice.
